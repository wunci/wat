## Install

```
npm install wat-ui -S
```

## Quick Start

```
import Vue from 'vue'
import Wat from 'wat-ui'
import 'wat-ui/lib/wat.css'

Vue.use(Wat)
```
## CDN

```
<link rel="stylesheet" href="https://unpkg.com/wat-ui/lib/wat.min.css">
```
